Create virtual env => pipenv --python 3.9
Where is env install => pipenv --venv
Where is python intepret install => pipenv --py
Run python inside of env => pipenv run python
Show env packages pipenv graph
Show run packages with list => pipenv run pip list
Uninstall package => pipenv uninstall numpy or pipenv run pip uninstall numpy
Remove pipenv => pipenv --rm
